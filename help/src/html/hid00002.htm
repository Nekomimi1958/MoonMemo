インストール

動作環境

Windows XP 以降なら多分だいじょうぶです。

インストールするファイル

適当なディレクトリに以下のファイルを置いてください。
また、バージョンアップの際は、ファイルを上書きしてください。

MoonMemo.exe月メモ本体
MoonMemo.chmこのヘルプファイル
koyomi.dll暦データ計算DLL(片貝充男氏作)

最初に起動したときには、デフォルトの基本データファイルとして MM_DATA.TXT が、またメモテキストファイルとして MM_MEMO.TXT が自動的に作られます。他のファイルに変更したい場合は、設定 - データ で行ってください。

また MoonMemo.INI(UTF-8) というファイルが作られ、設定情報等はそこに保存されます。レジストリは使用していません。

※V1.50 からデータファイルを Unicode(UTF-8) でも保存できるようになりました。読み込む際は、UTF-8、SjiftJIS のどちらでも開けます。UTF-8 で保存されたファイルは、以前のバージョンでは開けませんので注意してください。予定やメモで、ShiftJISにない文字を使ったり、多言語を混在させたい場合には、設定 - データ の「データをUnicode(UTF-8)で保存」をチェックしてください。
なお INI ファイルは、V1.50 から UTF-8 で保存されます。

※V1.25から、旧暦等の情報はDLLを用いて取得するようにしたため、今まで同梱していた旧暦情報ファイルは不要となりました。V1.25以前で設定されていた暦情報データファイルは起動時に一旦解除されます。ユーザ作成の暦情報データファイルを用いたい場合は、改めて設定しなおしてください。

source ディレクトリにはソースコード(C++Builder 10.4.1 用、Konopka Signature VCL Controls を使用)がありますので、自由にご利用ください。


アンインストール

インストールしたファイルおよび INIファイル等を削除して下さい。
